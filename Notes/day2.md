# Day 2 — Operators & Expressions

## Topics
- Arithmetic, Comparison, Logical, Identity, Membership operators
- Precedence & associativity
- Short-circuit logic
- Chained comparisons
- Expression evaluation

## Key Points
- Precedence: () > ** > unary > * / // % > + - > comparisons > is/in > and > or
- and stops at first False
- or stops at first True
- 3 ** 2 ** 2 = 3 ** (2 ** 2)
- Chained comparisons link logically: a < b < c

## Truths
- True and False or True → True
- Identity (is) compares memory, == compares value
