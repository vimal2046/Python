# Day 1 — Python Basics

## Topics
- Variables and objects
- Data types (int, float, bool, complex, str, None)
- type(), id()
- input()
- Basic input validation
- String slicing

## Key Points
- Everything from input() is a string
- Python variables are references to objects
- Use type() to check datatype
- Use id() to see memory reference
- Slicing format: s[start:end:step]

## Truths
- int() and float() convert strings but require valid formats
- .isdigit() checks digits only — no signs, no decimals
